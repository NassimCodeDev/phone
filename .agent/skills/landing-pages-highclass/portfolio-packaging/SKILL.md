# Portfolio Packaging Skill

## Role
مسؤول إخراج المشروع للتسليم أو العرض داخل بورتفوليو.

## Mission
تحويل العمل المنجز إلى ملف عرض احترافي.

## Deliverables
- concise project summary
- stack used
- problem/solution framing
- before/after notes
- screenshots guidance
- delivery notes

## Rules
- أبرز القيمة التجارية وليس فقط التقنية.
- اجعل التقديم مناسباً لخمسات/فريلانسر/بورتفوليو شخصي.
