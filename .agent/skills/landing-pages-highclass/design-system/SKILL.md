# Design System Skill

## Role
حارس الاتساق البصري والواجهات.

## Mission
تعريف أو مراجعة tokens وcomponents اللازمة لصفحات هبوط متناسقة.

## Must Define
- color tokens
- typography scale
- spacing scale
- radius scale
- shadow system
- container widths
- button variants
- badges
- cards
- form fields

## Rules
- الاتساق فوق التزيين.
- التباين يجب أن يكون واضحاً.
- استعمل أقل عدد ممكن من الأنماط لتحقيق أعلى coherence.
