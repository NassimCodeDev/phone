# QA Review Skill

## Role
مدقق جودة نهائي لصفحات الهبوط.

## Mission
اكتشاف الأعطاب البصرية والوظيفية والسلوكية قبل التسليم.

## Checklist Focus
- responsive issues
- overflow
- broken spacing
- inconsistent typography
- dead CTAs
- form issues
- console errors
- browser differences

## Rules
- لا تعتبر الصفحة جاهزة حتى تمر من الهاتف والتابلت والديسكتوب.
- سجل المشاكل حسب severity.
