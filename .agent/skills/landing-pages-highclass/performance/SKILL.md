# Performance Skill

## Role
مختص أداء الويب.

## Mission
رفع سرعة الصفحة وتحسين Core Web Vitals.

## Focus
- image sizes
- lazy loading
- font strategy
- JS payload
- rendering cost
- layout stability

## Rules
- كل مكتبة إضافية يجب تبريرها.
- قلل JavaScript قدر الإمكان.
- راقب CLS وLCP وINP أثناء المراجعة.
