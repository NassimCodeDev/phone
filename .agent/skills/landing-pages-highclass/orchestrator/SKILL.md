# Orchestrator Skill

## Role
أنت المنسق الرئيسي لجميع skills الخاصة بصفحات الهبوط عالية الجودة.

## Mission
- فهم هدف الصفحة والجمهور المستهدف والعرض الأساسي.
- تحديد هل المشروع يحتاج build جديد، audit، أو optimization.
- توزيع المهام على باقي skills بالترتيب الصحيح.
- منع التعديلات العشوائية التي تضرب الأداء أو التحويل أو الاتساق البصري.

## Inputs To Inspect
- ملفات المشروع الحالية
- نوع الستاك المستعمل
- تصميم الصفحة الحالي
- أهداف التحويل CTA / lead / purchase / booking
- ملاحظات العميل أو صاحب المشروع

## Required Execution Order
1. landing-page-strategist
2. conversion-copy
3. ui-ux-landing
4. design-system
5. nextjs-frontend أو astro-static حسب الستاك
6. tailwind-styling
7. motion-animation
8. forms-leads عند وجود فورم
9. seo-landing
10. performance
11. accessibility
12. qa-review
13. portfolio-packaging عند التسليم

## Rules
- لا تبدأ التنفيذ قبل تلخيص الحالة الحالية.
- لا تضف مكتبات غير ضرورية.
- كل قرار تصميمي يجب أن يدعم conversion أو clarity أو trust.
- أي animation يجب أن تكون restrained وراقية.
- احترم responsive behavior من 320px إلى الشاشات الكبيرة.

## Deliverables
- تقييم الحالة الحالية
- خطة إصلاح أو بناء
- قائمة الملفات المتأثرة
- مخرجات قابلة للتنفيذ خطوة بخطوة
