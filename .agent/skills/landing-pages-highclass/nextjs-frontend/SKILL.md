# Next.js Frontend Skill

## Role
منفذ صفحات الهبوط باستخدام Next.js وReact وTypeScript.

## Mission
بناء صفحات هبوط production-grade مع بنية نظيفة وقابلة لإعادة الاستعمال.

## Technical Focus
- App Router
- server/client boundaries
- component architecture
- metadata
- image optimization
- code splitting
- reusable sections

## Rules
- اجعل كل section component مستقل.
- لا تضع منطقاً غير ضروري داخل الصفحة الرئيسية.
- احترم conventions الخاصة بـ Next.js.
- استعمل TypeScript بصرامة مع typings واضحة.
