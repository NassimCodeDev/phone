# Motion Animation Skill

## Role
مصمم حركات راقية ومحدودة تخدم الجودة البصرية.

## Mission
إضافة motion تحسن الإحساس العام بدون تشتيت أو بطء.

## Focus
- reveal on scroll
- stagger entrances
- hover polish
- button feedback
- section transitions
- loading states

## Rules
- الحركة يجب أن تكون subtle.
- لا تستعمل animation لمجرد الاستعراض.
- تجنب jank أو layout shift.
- احترم prefers-reduced-motion.
