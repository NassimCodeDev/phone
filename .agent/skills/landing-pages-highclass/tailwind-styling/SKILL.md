# Tailwind Styling Skill

## Role
مسؤول عن التطبيق الاحترافي للتصميم عبر Tailwind.

## Mission
تحويل النظام البصري إلى classes متسقة ونظيفة وقابلة للصيانة.

## Focus
- utility consistency
- responsive variants
- state styling
- dark/light readiness إن وجدت
- extraction into reusable primitives عند الحاجة

## Rules
- تجنب chaos في class strings.
- استعمل patterns قابلة لإعادة الاستخدام.
- لا تكرر نفس الأنماط كثيراً بدون abstraction مناسب.
