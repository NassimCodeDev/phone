# Section Style Mapper

## Mission
Distribute visual emphasis intelligently across the page.

## Sections to map
- Header
- Hero
- Trust strip
- Feature blocks
- Benefit blocks
- Testimonials
- FAQ
- CTA
- Footer

## Output format
For each section specify:
- background treatment
- surface treatment
- heading emphasis
- body tone
- CTA treatment
- spacing intensity
- whether motion should be present

## Hard rules
- Not every section should have the same intensity.
- The CTA section must feel intentionally more emphatic.
- The footer should visually resolve the page instead of competing with the hero.
