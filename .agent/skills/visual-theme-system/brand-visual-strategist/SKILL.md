# Brand Visual Strategist

## Mission
Determine the correct visual mood before any colors, fonts, or section styling are implemented.

## Use this skill when
- The theme is still unclear.
- The design defaults to dark mode without justification.
- The page looks generic or visually disconnected from the offer.
- The visual identity needs to match the idea, audience, and conversion goal.

## Inputs to inspect
- Product or service category
- Audience sophistication
- Tone: luxury, corporate, calm, energetic, editorial, technical, friendly, serious
- Primary language and writing direction
- Content density
- Desired emotional effect
- Conversion goal

## Output format
Provide:
1. Recommended theme type: light, dark, mixed, soft-neutral, editorial, premium-corporate, luxury, bold-commercial, etc.
2. One-paragraph visual rationale.
3. Three mood keywords.
4. Design risks to avoid.
5. Recommendation for how visual emphasis should be distributed across hero, trust areas, cards, CTA, footer.

## Hard rules
- Never default to dark mode without contextual justification.
- Never choose a theme based on trend alone.
- The chosen visual mood must match the product, audience, and content density.
- If readability or trust would be harmed by a dark interface, reject dark mode.
- If the page is Arabic-first, account for typography density and visual breathing room.

## Rejection criteria
Reject the current direction if:
- It feels copied from a common SaaS template.
- It imposes dark mode where a lighter or mixed system would convert better.
- It creates a mismatch between brand tone and visual tone.
