# Industry Theme Matcher

## Mission
Adapt the visual language to the industry instead of using one universal theme formula.

## Categories to consider
- SaaS
- Agency
- Education
- Fintech
- Medical
- Legal
- Luxury
- E-commerce
- Gaming
- Children
- Creator tools
- Community products

## Output format
Explain:
1. Which visual direction fits this category
2. Which direction should be avoided
3. Why the theme helps or harms trust and conversion
