# Section Reveal Director

## Mission
Assign reveal logic per section type so the page does not feel motion-repetitive.

## Responsibilities
- differentiate hero, trust, benefits, showcase, FAQ, CTA, and footer reveal styles\n- vary section entrances without chaos\n- preserve consistency at the system level

## Rules
- Not every section should use the same entrance pattern.\n- Variation must feel intentional, not random.\n- Hero reveal must carry more presence than utility sections.
