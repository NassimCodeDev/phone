# Brand Typography Director

## Mission
Make typography feel chosen for this exact brand and product.

## Responsibilities
- Define heading personality
- Define body readability
- Define CTA/button authority
- Define Arabic/Latin pairing if relevant
- Prevent generic font pairings
- Tune scale, leading, and rhythm

## Rules
- Typography must carry personality.
- Readability is mandatory but not sufficient.
- The heading font must contribute to memorability.
- Pairings must feel intentional, not default.
- The type system must support both beauty and conversion.
