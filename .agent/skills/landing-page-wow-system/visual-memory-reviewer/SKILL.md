# Visual Memory Reviewer

## Mission
Judge whether the page contains memorable visual signatures that remain in the mind after viewing.

## Review Questions
- What would a user remember 10 minutes later?
- Is there a distinctive hero memory?
- Are the colors and type recognizable?
- Is the page visually ownable?

## Rules
- If the page is easy to forget, it fails.
- Generic premium polish without memorability is not enough.
