# Conversion Pressure Reviewer

## Mission
Review whether the page steadily and elegantly pushes the user toward action.

## Review Questions
- Is the CTA hierarchy clear?
- Do sections increase readiness to act?
- Is the value proposition reinforced enough?
- Are objections softened before the final CTA?

## Rules
- Weak CTA gravity is a fail.
- Decorative storytelling without commercial force is a fail.
