# Installation

1. Unzip the pack.
2. Copy `.agent/` and `.agents/` into the root of your project.
3. Optionally copy `prompts/` if you want ready-made execution prompts.
4. Start with `prompts/master-premium-landing-page-prompt.txt`.

## Folder summary
- `.agent/skills/ultimate-landing-page-system/` → specialist skills
- `.agents/rules/` → shared standards that all specialists should respect
- `.agents/workflows/` → reusable execution flows
