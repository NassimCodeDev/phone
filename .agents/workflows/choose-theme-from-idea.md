# Choose Theme From Idea Workflow

## Goal
Determine a suitable theme, color system, and typography system from the product idea before coding.

## Steps
1. Identify product category, audience, language, tone, and conversion goal.
2. Decide the visual mood and theme type.
3. Select a color direction and validate its psychology.
4. Select typography and validate Arabic/Latin pairing where needed.
5. Define tokens and section mapping.
6. Produce implementation-ready recommendations.
