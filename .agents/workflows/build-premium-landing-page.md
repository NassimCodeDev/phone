# Build Premium Landing Page

1. Define positioning and audience.
2. Choose visual direction and theme rationale.
3. Define typography and semantic color system.
4. Establish design tokens and layout rhythm.
5. Map sections according to conversion storytelling.
6. Write premium copy.
7. Implement reusable sections and UI primitives.
8. Add restrained motion and coherent buttons/icons.
9. Complete SEO, performance, and accessibility hardening.
10. Run final QA and polish pass.
