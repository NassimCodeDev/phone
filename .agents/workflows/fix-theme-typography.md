# Fix Theme and Typography Workflow

## Goal
Correct weak or generic colors, typography, and section styling in an existing landing page.

## Steps
1. Audit the current page and identify:
   - unjustified dark mode
   - generic palettes
   - poor Arabic or Latin typography
   - weak font pairing
   - contrast issues
   - inconsistent section treatment
2. Use brand-visual-strategist to decide the appropriate visual mood.
3. Use industry-theme-matcher and color-psychology-agent to validate the direction.
4. Use color-strategy-agent to define a better palette.
5. Use typography-strategist, arabic-rtl-typography-agent, and font-pairing-agent to fix font choices.
6. Use theme-token-system-agent to convert decisions into reusable tokens.
7. Use section-style-mapper to distribute visual emphasis across the page.
8. Use readability-accessibility-agent to validate legibility and accessibility.
9. Use anti-generic-design-agent and design-polish-reviewer to premiumize the result.
10. Apply the fixes directly and summarize the final visual system.
