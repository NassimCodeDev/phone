# Audit Existing Landing Page

1. Inspect the current product/offer clarity.
2. Audit section order and storytelling.
3. Audit copy quality and CTA hierarchy.
4. Audit theme, color system, typography, and RTL/Arabic readiness.
5. Audit motion, buttons, icons, forms, and interaction states.
6. Audit SEO, performance, accessibility, and responsiveness.
7. Produce a prioritized weakness list.
8. Apply fixes directly, starting with the most conversion-critical items.
