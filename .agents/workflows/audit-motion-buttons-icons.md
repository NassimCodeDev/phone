# Audit Motion, Buttons, and Icons

1. Inspect current motion patterns, button variants, and icon usage.
2. Identify inconsistency, over-animation, weak CTA hierarchy, and icon mismatch.
3. Define the corrected interaction philosophy.
4. Apply fixes directly in code.
5. Run accessibility checks:
   - reduced motion
   - focus visibility
   - keyboard navigation
   - contrast on states
6. Deliver audit summary + files modified + remaining optional polish.
