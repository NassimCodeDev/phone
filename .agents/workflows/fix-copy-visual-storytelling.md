# Workflow: Fix Copy + Visual Identity + Section Storytelling

1. Audit the current landing page copy, structure, and visual identity.
2. Detect mismatches between:
   - brand tone and colors
   - typography and message level
   - section order and persuasion flow
   - CTA wording and audience readiness
3. Define:
   - audience
   - awareness stage
   - emotional tone
   - promise
   - trust strategy
   - storytelling arc
4. Redesign the section flow if needed.
5. Rewrite the copy with premium, specific, conversion-aware language.
6. Align headlines, CTA emphasis, and trust language with the visual direction.
7. Improve scannability, hierarchy, and section transitions.
8. Run a final anti-generic review and polish pass.
