# Initialize Next.js Premium Landing Workflow

Use this workflow only when the workspace contains no usable frontend foundation.

## Initialize
- create Next.js project
- enable TypeScript
- use App Router
- use src directory
- prepare reusable sections/components
- prepare design token layer

## Supporting libraries
- Tailwind CSS
- lucide-react
- motion / framer-motion only if justified

## Constraints
- keep the foundation minimal
- keep the structure scalable
- avoid adding unnecessary dependencies
- prepare for premium landing-page implementation
