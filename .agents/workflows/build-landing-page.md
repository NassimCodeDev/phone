# Build Landing Page

Use the orchestrator skill to create a new premium landing page from scratch.

Execution order:
1. landing-page-strategist
2. conversion-copy
3. ui-ux-landing
4. design-system
5. nextjs-frontend or astro-static
6. tailwind-styling
7. motion-animation
8. forms-leads if needed
9. seo-landing
10. performance
11. accessibility
12. qa-review

Required output:
- Final page structure
- Components list
- Copy blocks
- Implementation notes
- QA completion notes
