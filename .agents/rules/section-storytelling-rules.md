# Section Storytelling Rules

- Each section must justify its presence through one of: clarity, desire, trust, objection handling, proof, or action.
- Do not add filler sections.
- Hero must communicate positioning, value, and next action quickly.
- Social proof must reduce perceived risk.
- Features must translate into benefits.
- Process sections must reduce complexity and fear.
- FAQ must remove hesitation, not repeat sales copy.
- Final CTA must feel stronger than the first CTA, not repetitive.
- Visual emphasis should follow narrative emphasis.
