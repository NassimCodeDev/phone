# Copy + Visual Identity Standards

- Never write copy independently from the visual identity.
- Headlines, typography feel, and color emphasis must support the same emotional direction.
- Never default to generic startup language.
- Avoid inflated claims unless the product proof supports them.
- Each section must have one clear narrative role.
- Every CTA must match the awareness stage and user intent.
- Use short, scannable copy blocks for landing pages.
- Maintain consistency of tone across hero, features, testimonials, FAQ, and final CTA.
- Arabic copy must sound natural, persuasive, and modern—not literal or rigid.
- If bilingual, Arabic and English must feel like the same brand voice.
