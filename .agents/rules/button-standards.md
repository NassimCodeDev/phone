# Button Standards

- Maintain a coherent button taxonomy: primary, secondary, tertiary/ghost, destructive, link.
- Buttons must have explicit states: default, hover, focus-visible, active, disabled, loading where relevant.
- Primary CTA must have the clearest affordance on the page.
- Padding, radius, border weight, font size, and icon spacing must follow a consistent token system.
- Button labels must be action-oriented and commercially clear.
