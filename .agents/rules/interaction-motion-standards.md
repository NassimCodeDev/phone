# Interaction Motion Standards

- Motion must be purposeful, restrained, and hierarchy-aware.
- Prefer opacity + slight translate + subtle scale over flashy composite effects.
- Default duration guidance:
  - fast micro interactions: 120ms to 180ms
  - standard UI transitions: 180ms to 240ms
  - section reveals: 300ms to 500ms
- Prefer smooth standard easings over novelty easings.
- Respect reduced-motion preferences at all times.
- Do not animate layout in a way that causes instability or delayed readability.
